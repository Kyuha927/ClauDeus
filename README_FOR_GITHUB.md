# ClauDeus: Multi-Model AI Messaging Workflow

이 리포지토리는 안티그래비티(Antigravity)를 활용하여 여러 AI 모델(Gemini, Sonnet, Opus)을 마치 메시징 앱처럼 편리하게 부려먹을 수 있는 워크플로우 템플릿입니다.

## 🌟 주요 특징
- **Zero Configuration**: 별도의 라이브러리 설치나 복잡한 설정 없이 즉시 동작합니다.
- **최고 사양 모델**: 항상 최신 사양 모델(Opus 4.6, Sonnet 4.5, Gemini 3 Pro High)을 사용하도록 최적화되어 있습니다.
- **메시징 경험**: 파일을 저장하면 AI가 즉시 작업을 시작하고 결과를 업데이트합니다.
- **이력 관리**: 모든 대화 내역이 `HISTORY.md`에 타임스탬프와 함께 자동 기록됩니다.

## 🚀 시작하기
1. 이 리포지토리를 다운로드(또는 Clone)합니다.
2. Windows 사용자는 `setup.ps1`을 실행하여 폴더 구조를 준비합니다.
3. 해당 폴더를 **안티그래비티(Antigravity)**로 엽니다.
4. 채팅창에 **`ㄱ`**을 입력하여 초기 동기화를 완료합니다.

## 🛠️ 사용 방법
1. `prompts/` 폴더 내의 파일들을 확인하고 내용을 수정해 보세요.
2. 에디터의 **Auto Save**를 켜두면, 글을 쓰는 즉시 AI가 일하기 시작합니다.
3. **[RESULTS_SUMMARY.md](RESULTS_SUMMARY.md)**에서 최신 결과를 확인하고, **[HISTORY.md](HISTORY.md)**에서 전체 이력을 확인하세요.
